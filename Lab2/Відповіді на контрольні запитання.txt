Контрольні питання
1.	Який метод введення даних користувачем із використанням модальних вікон повертає значення true або false?
2.	Поясніть який результат ми отримаємо, якщо перенесемо скрипт confirm.js  у заголовок HTML-документа?
3.	Яке призначення методу prompt( )?
4.	Пояснити призначення та відмінності методів textContent та innerHTML.
5.	Яке призначення методу confirm( )?

1. Метод confirm() введення даних із використанням модальних вікон повертає значення true або false.
2. Якщо перенести скрипт confirm.js у заголовок HTML-документа, то під час відкриття сторінки одразу з'явиться модальне вікно з повідомленням "Готові гризти граніт науки?".
3. Метод prompt() використовується для введення даних користувачем у текстове поле і повертає введене значення або null.
4. Метод textContent встановлює або повертає текстовий вміст елемента, тоді як innerHTML встановлює або повертає HTML-вміст елемента.
5. Метод confirm() використовується для відображення модального вікна з повідомленням і двома кнопками - "OK" і "Скасувати". Він повертає true, якщо натиснуто кнопку "OK", і false, якщо натиснуто кнопку "Скасувати".
